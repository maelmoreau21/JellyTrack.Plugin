JellyTrack

Configure la connexion vers votre serveur JellyTrack.

URL cible exacte. Exemples : http://192.168.1.100:3000/api/plugin/events ou http://192.168.1.100:3000/api/webhook/jellyfin. Si vous mettez seulement l'hôte, /api/plugin/events est utilisé par défaut.
La clé API générée dans les paramètres de JellyTrack (format: jt_xxxxxxxxxxxx)
Intervalle entre chaque heartbeat envoyé à JellyTrack (défaut: 60 secondes)
Intervalle entre chaque mise à jour de progression de lecture (défaut: 15 secondes)
